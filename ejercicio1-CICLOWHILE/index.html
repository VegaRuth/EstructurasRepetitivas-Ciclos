Ejercicio #1: SUMA 10 NUMEROS CICLO WHILE

Se requiere un algoritmo para obtener la suma de diez cantidades mediante la utilización de un ciclo “WHILE”. Realice el diagrama de flujo y el pseudocódigo

Pseudocódigo

Variables
Nombre Descripción Tipo
n Primer valor Float
i Valor a incrementar Float


Proceso

  1. Inicio
  2. Leer "n" e "i"
  3. Revisar condicion: Si "n" < 10, entonces
  4. Ciclo
    Incrementa n en uno cada vez(n++); asignalo a i y suma el nuevo valor de n cada vez.(i+=n);

    Si no finaliza el ciclo

  5. Fin
