Ejercicio #6: TRIANGULO

Un triangulo rectángulo puede tener lados que sean todos enteros. El conjunto de tres valores enteros para los lados de un triángulo rectángulo se conoce como una terna pitagórica. Estos tres lados deben satisfacer la relación de que la suma de los cuadrados de dos lados es igual al cuadrado de la hipotenusa. Encuentre todas las ternas de Pitágoras para el cateto opuesto, cateto adyacente e hipotenusa, todos ellos no mayores de 500.

Pseudocódigo

Variables
Nombre Descripción Tipo
numero_personas Primer valor a multiplicar Int
presupuesto Resultado de operación Int


Proceso

  1. Inicio
  2. Leer "numero_personas"
  3. Condición

    Si "numero_personas" es < 200, entonces "presupuesto" es = "numero_personas"* 95, alerta " el presupuesto es de ""presupuesto";

    Y si "numero_personas" es > 200 && < 300 , entonces "presupuesto" es = "numero_personas"*85, alerta " el presupuesto es de ""presupuesto";

    Si no, entonces: "presupuesto" es = "numero_personas"*75, alerta " el presupuesto es de ""presupuesto";
  4. Fin
